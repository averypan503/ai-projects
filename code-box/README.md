# CodeBox - Full-Featured Web IDE

## 📖 项目简介
CodeBox 是一款基于浏览器原生能力构建、对标 VSCode 完整能力体系的全功能前端集成开发环境。项目采用纯前端架构，无任何后端服务依赖，具备轻量化、高性能、高可扩展的特性，内置完整的标准化开发工作流，涵盖文件管理、代码编辑、语法校验、项目运行调试、版本控制、终端交互、插件拓展等全栈开发能力。

本项目适配前端项目开发、小型全栈项目迭代、在线代码编辑与学习等各类开发场景，依托标准化、模块化的现代化前端架构，具备极强的可拓展性与可维护性，是一套生产级别的浏览器端完整 IDE 解决方案。项目整体开发由 AI 辅助完成架构设计、功能落地与工程化优化。

## 🛠 技术栈
- **核心框架**：React 18 + Hooks
- **状态管理**：Redux Toolkit
- **编辑器内核**：Monaco Editor
- **本地文件交互**：Browser File System API
- **样式解决方案**：SCSS
- **开发辅助**：AI 辅助架构设计、功能开发、工程化迭代

## ⚙️ 技术实现
本模块记录项目各功能模块的标准化技术实现方案，可随项目功能迭代持续更新优化。

### 1. 工作区与文件系统模块
基于浏览器原生 File System API 实现本地文件夹挂载与权限管理，通过全局句柄统一管理工作区目录权限，实现本地文件、文件夹的标准化增删改查能力。通过递归遍历解析本地目录结构，生成标准化树形文件数据，结合 Redux 实现全局状态统一管理，完成前端视图与本地磁盘数据双向实时同步，支撑上层所有 IDE 功能的文件读写需求。

### 2. 视图与布局模块
采用组件化分层设计，拆分顶部导航栏、文件树视图、编辑区、控制台面板等核心模块。通过 Hooks 管理组件局部状态，结合全局 Redux 状态实现多组件数据联动，实现文件标签页管理、文件夹展开收起、文件激活高亮、操作日志提示等基础交互能力，整体布局适配 IDE 标准交互逻辑。

## 📌 项目功能规划
本项目以 VSCode 级全功能 Web IDE 为核心定位，通过模块化分层设计，覆盖完整的浏览器端开发全链路能力，所有规划为项目既定核心功能体系。

### 阶段一：IDE 基础底座能力
实现 IDE 必备的底层支撑能力，构建完整的项目工作区运行体系。
- 标准化项目架构搭建，完成全局状态分层设计与模块化拆分
- 本地工作区挂载、文件夹授权、目录遍历与解析能力
- 文件树可视化渲染、多标签页管理、全局日志系统搭建
- 本地文件与文件夹完整 CRUD 能力，保障磁盘与状态双向同步

### 阶段二：专业代码编辑能力
集成专业编辑器内核，实现标准化、高适配的代码编辑交互体系，对齐主流 IDE 交互标准。
- 集成 Monaco Editor 编辑器内核，适配多语言开发场景
- 语法高亮、代码折叠、智能缩进、行号展示等基础编辑能力
- 接入 LSP 语法校验，实现代码智能提示、错误检测、代码补全
- 代码格式化、批量注释、代码重构、编辑历史回溯功能
- 文件自动保存、手动保存、内容缓存恢复机制

### 阶段三：项目运行与实时预览能力
搭建浏览器端项目编译、运行与热更新体系，支撑前端项目全流程开发调试。
- HTML、CSS、JavaScript 实时预览与即时运行
- TypeScript 实时编译、语法检测与错误热提示
- 项目热更新机制，文件修改后实时刷新预览页面
- 运行日志捕获体系，统一收集控制台输出与运行报错信息

### 阶段四：Git 版本控制能力
内置完整可视化 Git 体系，实现网页端一站式版本控制管理能力。
- Git 仓库初始化、工作区文件状态监听、变更检测
- 代码提交、拉取、推送、分支切换、合并等核心 Git 操作
- 可视化 Diff 对比视图，支持文件变更细节预览
- 远程仓库关联、代码同步、版本回溯能力

### 阶段五：终端与代码调试体系
集成专业 IDE 必备的终端与调试功能，完善项目开发与问题排查能力。
- Web 终端模拟器，支持标准命令行输入输出
- npm、yarn、pnpm 等包管理命令执行与项目脚本运行
- 前端代码断点调试、变量监视、调用栈查看能力
- 运行时错误捕获、源码定位与异常信息展示

### 阶段六：个性化定制与 UI 生态能力
提供全方位自定义配置体系，适配各类开发者的使用习惯与视觉需求。
- 深浅色主题切换、自定义主题配色配置
- 字体大小、行高、缩进格式、快捷键自定义配置
- 面板拖拽拆分、工作区布局自定义保存与恢复
- 用户全局配置持久化存储，页面刷新配置不丢失

### 阶段七：插件拓展生态能力
搭建标准化插件体系，实现 IDE 功能的多元化拓展与自定义适配。
- 插件加载机制与完整生命周期管理逻辑
- 语法拓展、代码片段、主题插件、工具插件接入支持
- 自定义命令、自定义面板拓展能力
- 第三方插件接入规范，完善插件生态体系

### 阶段八：性能与工程化能力
适配大型项目开发场景，保障 IDE 的高性能、高稳定性与规范性。
- 优化大目录、大文件渲染逻辑，解决大容量项目卡顿问题
- 优化组件重渲染逻辑，减少内存占用，提升运行流畅度
- 浏览器兼容性适配与权限降级方案
- 统一代码规范，实现模块解耦与标准化工程化架构

## 📝 使用指南
1. 克隆项目至本地环境，安装依赖并启动开发服务
2. 通过项目工作区入口，选择本地目标开发文件夹
3. 授予浏览器文件夹读写权限，加载本地项目目录结构
4. 通过可视化文件树管理项目文件，使用编辑器完成代码开发工作

## ⚠️ 注意事项
- 项目基于现代浏览器原生标准 API 开发，仅兼容 Chrome、Edge 等主流最新版本浏览器
- 本地文件读写、目录管理等功能，需用户主动授予浏览器文件夹访问权限
- 项目所有文件操作直接作用于本地磁盘，操作项目文件及目录时请谨慎操作
